Job Portal System

A web-based Job Portal System developed as part of a web development internship project. This application allows users to register, log in, browse available jobs, search for jobs, and apply directly through a user-friendly dashboard.

Features

-  User Registration & Login System
-  Dashboard with Job Listings (Card UI)
-  Search Jobs Functionality
-  Apply for Jobs
-  Logout with Session Management
-  Responsive UI using Bootstrap

Technologies Used

- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL (phpMyAdmin)
- Server: XAMPP
- Code Editor: VS Code

How to Run the Project

1. Install XAMPP and start Apache & MySQL
2. Copy the project folder into htdocs
3. Open phpMyAdmin and create database `job_portal`
4. Import required tables (users, jobs, applications)
5. Run the project in browser:
   http://localhost/job_portal
